# 🧩 csCubeSOLVER Professional

![Status](https://shields.io)
![License](https://shields.io)

O **csCubeSOLVER** é um timer de alta performance para speedcubing, inspirado no csTimer, mas com um design refinado e funcionalidades exclusivas de duelo contra inteligência artificial.

## 🚀 Funcionalidades

- **Multi-Cubo:** Suporte para 2x2, 3x3, 4x4 e Pyraminx com algoritmos de embaralhamento reais.
- **Bot Versus Mode:** Dispute contra um bot com dificuldades ajustáveis (Iniciante até God Mode).
- **Estatísticas Avançadas:** Cálculo automático de Ao5 (Média de 5) e Ao12 (Média de 12) seguindo as regras oficiais da WCA.
- **Design Premium:** Interface em tons de Branco e Azul (Ocean Blue) focada em legibilidade e performance.
- **Exportação:** Salve seus recordes em formato CSV para análise externa.

## 📂 Estrutura do Projeto

*   `index.html` - O timer principal com estatísticas e histórico.
*   `Botversus.html` - A arena de duelo contra o Bot Cyberzin.
